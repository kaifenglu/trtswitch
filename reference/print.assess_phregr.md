# Print method for assess_phregr objects

Prints the concise information of an assess_phregr fit.

## Usage

``` r
# S3 method for class 'assess_phregr'
print(x, ...)
```

## Arguments

- x:

  An object of class `assess_phregr`.

- ...:

  Ensures that all arguments starting from "..." are named.

## Value

A printout from the fit of an assessment of proportional hazards
assumption of a Cox model.
