# Print method for ipcw objects

Prints the concise information of an ipcw fit.

## Usage

``` r
# S3 method for class 'ipcw'
print(x, ...)
```

## Arguments

- x:

  An object of class `ipcw`.

- ...:

  Ensures that all arguments starting from "..." are named.

## Value

A printout from the fit of an inverse-probability of censoring weights
model.
