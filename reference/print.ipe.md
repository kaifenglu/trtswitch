# Print method for ipe objects

Prints the concise information of an ipe fit.

## Usage

``` r
# S3 method for class 'ipe'
print(x, ...)
```

## Arguments

- x:

  An object of class `ipe`.

- ...:

  Ensures that all arguments starting from "..." are named.

## Value

A printout from the fit of an iterative parameter estimation.
