# Print method for rpsftm objects

Prints the concise information of a rpsftm fit.

## Usage

``` r
# S3 method for class 'rpsftm'
print(x, ...)
```

## Arguments

- x:

  An object of class `rpsftm`.

- ...:

  Ensures that all arguments starting from "..." are named.

## Value

A printout from the fit of a rank-preserving structural failure time
model.
